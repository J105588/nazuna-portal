Development Log

なずなポータルの更新履歴と開発状況

v1.2.0 2025.11.23

PWA対応とプッシュ通知機能の実装

iOSおよびAndroid端末でのアプリのような操作感を実現しました。

  • Feature iOS/Androidのホーム画面への追加に対応（manifest.json）
  • Feature Firebaseを使用したプッシュ通知機能の基盤追加
  • Fix iOS Safariでのステータスバー表示の調整
  • Maint ファビコンおよびアプリアイコンの高解像度化
v1.1.5 2025.11.15

フォーラム機能の不具合修正

  • Fix コメント投稿時に画面がリロードされる問題を修正
  • Fix 一部のユーザーアイコンが表示されないバグを修正
  • Maint Supabaseへの接続処理の最適化
v1.1.0 2025.11.01

「なずなフォーラム」ベータ版公開

生徒同士で意見交換ができる掲示板機能をリリースしました。

  • Feature 新規スレッド作成機能の実装
  • Feature コメント機能および「いいね」ボタンの実装
  • Feature 不適切な投稿の自動フィルター機能（β版）
v1.0.0 2025.10.20

なずなポータル サイトオープン

市川学園生徒会の公式サイトとして正式リリースしました。

  • Feature トップページ、生徒会紹介、部活動一覧ページの公開
  • Feature 全校アンケート機能の実装
